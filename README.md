# WIP - WEBPACK STARTER

Webpack starter for developing static html/js/sass. Work in app directory the run webpack to compile to `dist/` folder


### Pre-requisites
* webpack global - `npm -g webpack`

### Get Started

`git clone`

work in app directory

`webpack`
